minesweeper sample webservice

This is a toy webservice for a hypothetical factory of "minesweeper" games.
The idea is that given a minefield, the service creates the hints for it and either returns them directly or stores them for later retrieval.

To submit a minefield, POST a payload giving the dimensions of the minefield, followed by the field itself. Empty spaces are marked as '.', mines are marked as '*'.
For example the following POST payload: 

{
    "height": 5,
    "width": 4,
    "minefield" : [
        "....",
        ".*..",
        "....",
        "..*.",
        "...."
    ]
}
Would yield the following result: 
[
    "1110",
    "1*10",
    "1221",
    "01*1",
    "0111"
]
